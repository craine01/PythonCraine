This repository documents my learning path in Python programming, focusing on the fundamentals of computational thinking, control structures, and problem-solving. The journey emphasizes both theoretical understanding and hands-on application, preparing me for more advanced topics in software development and computer science.

## 🗺️ Learning Roadmap
1. **Introduction to Python**
   - What is Python? Why learn it?
   - Writing your first program (`print("Hello, World!")`)
   - Understanding syntax and indentation
   - Debugging basics

2. **Data, Variables, and Operations**
   - Variables and constants
   - Data types: integers, floats, strings, booleans
   - Operators: arithmetic, relational, logical
   - Input/output with `input()` and `print()`

3. **Control Structures**
   - Decision making with `if`, `elif`, `else`
   - Nested conditions
   - Looping and repetition:
     - `while` loops
     - `for` loops
   - Common logical errors and debugging strategies

4. **Recursion**
   - Concept of divide-and-conquer
   - Writing recursive functions
   - Examples: factorial, Fibonacci sequence
   - Understanding base cases and stack behavior

5. **Functions and Modular Thinking** *(upcoming)*
   - Why use functions?
   - Parameters and return values
   - Scope and lifetime of variables
   - Designing reusable modules

6. **Data Structures** *(future topics)*
   - Lists, tuples, and dictionaries
   - Iteration and manipulation
   - Searching and sorting basics
